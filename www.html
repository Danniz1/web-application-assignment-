What is the World Wide Web (WWW)?

The World Wide Web (WWW) is a system of interlinked hypertext documents accessed via the Internet. Invented by Tim Berners Lee in 1989 at CERN.

You can view web pages using a browser. The web uses HTTP to transfer pages, HTML to structure content, and URLs to locate resources.

Key facts:

  • The web is NOT the same as the Internet - it's a service that runs on the Internet
  • Web pages contain text, images, videos, and links to other pages
  • Hyperlinks allow you to navigate between pages by clicking
  • There are billions of web pages on the World Wide Web

Related terms: Internet, HTTP, HTML, URL
