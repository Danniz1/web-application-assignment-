What is HTTP?

HTTP stands for HyperText Transfer Protocol. It is the foundation of data communication on the World Wide Web. HTTP defines how messages are formatted and transmitted between web browsers and web servers.

When you click a link, your browser sends an HTTP request to the server, which then sends back an HTTP response containing the requested page.

Key facts:

  • HTTP is a request-response protocol (client asks, server answers)
  • Common HTTP methods: GET (retrieve data), POST (submit data), PUT (update), DELETE
  • HTTPS is the secure version using encryption (SSL/TLS)
  • HTTP response status codes: 200 (OK), 404 (Not Found), 500 (Server Error)

Related terms: Web Browser, Web Server, URL, TCP/IP
