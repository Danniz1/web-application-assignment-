Welcome

This website covers key Internet terminology used in Web Application Development. Click any term in the navigation bar to learn more about it.

Topics Covered

  • Internet - The global network of computers
  • World Wide Web (WWW) - The web of hyperlinked documents
  • TCP/IP - The core communication protocols
  • Web Browser - Software to access web pages
  • Web Server - Computers that host websites
  • URL - Uniform Resource Locator
  • DNS - Domain Name System
  • HTTP - HyperText Transfer Protocol
  • Intranet - A private internal network
  • Extranet - A controlled private network
  • Multitier Architecture - Layered system design
  • FTP - File Transfer Protocol
  • HTML - HyperText Markup Language
  • Web 1.0 - The first era of the web
  • Web 2.0 - The social and interactive web
  • Web 3.0 - The semantic web
  • Web 4.0 - The intelligent web
