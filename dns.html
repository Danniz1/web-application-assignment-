What is DNS?

DNS stands for Domain Name System. It's like the phonebook of the Internet - it translates human-friendly domain names (like google.com) into IP addresses (like 142.250.190.46) that computers use to identify each other.

Without DNS, you would have to remember long strings of numbers for every website you want to visit.

Key facts:

  • DNS servers are distributed worldwide for speed and reliability
  • When you type a domain, your browser asks a DNS resolver to find the IP address
  • DNS uses caching to speed up repeated lookups
  • Top-level domains include .com, .org, .net, .edu, and country codes like .ke

Related terms: Internet, URL, TCP/IP
