What is HTML?

HTML stands for HyperText Markup Language. It is the standard markup language used to create web pages and web applications.

HTML uses tags (like `

`, `

`, ``) to structure content - telling the browser what is a heading, a paragraph, an image, etc.

Key facts:

  • HTML is not a programming language - it's a markup language
  • The latest version is HTML5
  • Browsers read HTML and display it as web pages
  • HTML works together with CSS (for styling) and JavaScript (for interactivity)

Related terms: WWW, Web Browser, HTTP
