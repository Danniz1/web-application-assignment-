What is a URL?

URL stands for Uniform Resource Locator. It is the web address used to locate a specific resource (like a web page, image, or file) on the Internet.

A typical URL looks like: https://www.example.com/page.html

Parts of a URL:

  • Protocol: http:// or https:// (how to access the resource)
  • Domain name: www.example.com (which server to contact)
  • Path: /page.html (specific file or location)
  • Optional: port number, query parameters, and anchor (#section)

Related terms: HTTP, DNS, WWW
